## v1

### v1.0.0

**Released:** 2026-03-27

#### Added

- Initial release of AI Image Renamer.
- Automatic renaming for newly uploaded images using Groq Vision.
- Optional alt text generation for uploaded attachments.
- Support for JPEG, PNG, WebP, and GIF uploads.
- API key handling via the settings screen or `wp-config.php`.
- Encrypted storage for saved API keys and admin diagnostics.
