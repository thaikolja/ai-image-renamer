# Download

## WordPress.org (Recommended)

The easiest way to install AI Image Renamer is directly from the WordPress plugin directory:

👉 [**Download from WordPress.org**](https://wordpress.org/plugins/ai-image-renamer/)

Or search for **"AI Image Renamer"** in your WordPress admin under **Plugins → Add New**.

## GitHub

You can download the latest release or clone the repository from GitHub:

👉 [**GitHub Releases**](https://github.com/thaikolja/wp-ai-image-renamer/releases)

```bash
git clone https://github.com/thaikolja/wp-ai-image-renamer.git
cd wp-ai-image-renamer
composer install --no-dev --optimize-autoloader
```

## GitLab

A mirror of the repository is also available on GitLab:

👉 [**GitLab Repository**](https://gitlab.com/thaikolja/wp-ai-image-renamer)

## Requirements

Before installing, make sure your environment meets the following requirements:

| Requirement | Minimum Version |
|-------------|-----------------|
| WordPress | 6.0 |
| PHP | 8.2 |
| Groq API Key | [Get one here](https://console.groq.com/keys) |

::: tip
The Groq API offers a generous free tier. Registration takes less than a minute.
:::
